## Basic Commands

##### Connection syntax
```Ruby
ActiveRecord::Base.establish_connection ({
  adapter: 'sqlite3',
  database: 'db/pokemon.db'
  })
```

##### Required Gems
```Ruby
gem "pry"
gem 'sqlite3'
gem 'rake'
gem 'sinatra-activerecord'
gem 'require_all'
```
##### Active Record Logger for SQL Commands
```Ruby
ActiveRecord::Base.logger = Logger.new(STDOUT)
```
